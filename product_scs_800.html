Straight Cargo System 800

The Straight Cargo System 800 is a perfect solution for unbalanced cargo. Unlike the 700 model, it provides top support and is useful on vehicles with bumper space. The Straight Cargo System is great for ramp applications and is an innovative solution to unstable cargo carriers. It can be useful for all types of cargo carriers such as luggage baskets, wheelchair cargo carriers, receiver hitch cargo carriers, ATV and dirt bike carriers. In addition, this cargo carrier attachment can be useful for folding carriers, utility carriers, hunting gear carriers, outdoorsman carriers and camping gear carriers. If you have any form of trailer, cart or cargo carrier odds are the Straight Cargo System will be useful.
