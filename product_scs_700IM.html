Straight Cargo System 700IM

If your vehicle cargo keeps tilting and shifting, the Straight Cargo System is perfect for you. The Straight Cargo System 700IM provides bottom support for cargo carriers. Similar to the 700 model, the 700IM model also provides bottom support but with a slightly different fit. The 700IM model is used for cars lacking enough bumper space for the 800. The Straight Cargo System is perfect for ramp applications and is an innovative solution to unbalanced cargo. It can be useful for all types of cargo carriers such as luggage baskets, wheelchair cargo carriers, receiver hitch cargo carriers, ATV and dirt bike carriers. In addition, this cargo carrier attachment can be useful for folding carriers, utility carriers, hunting gear carriers, outdoorsman carriers and camping gear carriers. If you have any form of trailer, cart or cargo carrier odds are the Straight Cargo System will be useful.
